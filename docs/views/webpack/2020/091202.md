现在，我们可以把前面学习的webpack配置统一配置一下，生成一个基础的webpack开发环境配置。

## 项目目录

```bash
开发环境配置
    -src
        -css
            //css资源
        -imgs
            //图片资源
        -js
            --index.js
        -media
            //其他资源
        --index.html
    --webpack.config.js
```

## webpack.config.js

```bash
module.exports = {
    entry: './src/js/index.js',
    output: {
        filename: 'js/build.js',
        path: resolve(__dirname, 'build')
    },
    module: {
        rules: [
            {
                test: /\.less$/,
                use: ['style-loader', 'css-loader', 'less-loader']
            },
            {
                test: /\.css$/,
                use: ['style-loader', 'css-loader']
            },
            {
                test: /\.(jpg|png|gif)$/,
                loader: 'url-loader',
                options: {
                    limit: 8 * 1024,
                    name: '[hash:10].[ext]',
                    exModule: false,
                    outputPath: 'imgs'
                }
            },
            {
                test: /\.html$/,
                loader: 'html-loader'
            },
            {
                exclude: /\.(html|css|js|less|jpg|png|gif)/,
                loader: 'file-loader',
                options: {
                    name: '[hash:10].[ext]',
                    outputPath: 'media'
                }
            }
        ],
        plugins: [
            new HtmlWebpackPlugin({
                template: './src/index.html'
            })
        ],
        mode: 'development',
        devServer: {
            contentBase: resolve(__dirname, 'build'),
            compress: true,
            port: 3000,
            open: true
        }
    }
}
```
## 说明

开发环境只需要代码能够运行即可，我们使用 `webpack` 打包输出结果，使用 `npx webpack-dev-server` 只会在内存中编译打包，没有输出。
